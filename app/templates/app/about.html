{% trans "What is LwimiLinks?" %}

{% blocktrans trimmed %} LwimiLinks is a place for multilingual terminology and other useful language resources. We encourage institutions to upload their resources and register information on related projects. These resources can then be disseminated through the site to students, language practitioners, academics, linguists, lexicographers, and other users. {% endblocktrans %}

{% trans "Background information" %}

{% blocktrans trimmed %} The South African Centre for Digital Language Resources (SADiLaR) is a national research infrastructure that aims to ensure a digital future for our official languages. It supports researchers in the fields of digital humanities and social sciences and also assists institutions in their language implementation plans. {% endblocktrans %} {% trans "Read more about SADiLaR." %}

{% blocktrans trimmed %} SADiLaR conducted an audit at South African public higher education institutions that highlighted areas that require attention. {% endblocktrans %} {% trans "Read the report." %}

{% blocktrans trimmed %} LwimiLinks, developed by SADiLaR, is a platform which highlights previous and current work at institutions to encourage collaboration and disseminate resources. {% endblocktrans %} {% blocktrans with url="https://github.com/SADiLaR/term_platform/" trimmed %} The code of this web platform is available on GitHub. {% endblocktrans %}
